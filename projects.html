TO-DO-LIST SYSTEM

A console program made in C++ Language that creates, reads, updates, and deletes tasks. You can mark it as done or edit the task name and due date. This was used as a demo for the CRUD System in CC-103.




COIN CHANGE COUNTER

A C++ console program initially made in C language for the CS50x Week 1 "Cash" Problem Set. By using loops and a single if-else statement, it outputs the minimum American coins needed for change.

ELECTRIC BILL CALCULATOR

This C++ program calculates the electricity bill based on the number of kilowatt-hours (kWh) used. The user inputs the kWh consumed, and the program determines the cost per kWh based on predefined rates. It then calculates and displays the total electricity bill. This program demonstrates basic input/output operations, conditional statements, and arithmetic calculations in C++.




SUM QUOTIENT PROGRAM

This C++ program prompts the user to enter ten numbers, stores them in an array, and calculates the sum of all the numbers. Additionally, it computes the quotient of each number with the previous one if the current number is greater. The results, including the total sum and the final quotient, are displayed to the user. This program demonstrates array manipulation, input/output operations, and conditional arithmetic in C++.




MONTH CONVERTER

This C++ program converts a numerical input representing a month into its corresponding month name. The user inputs a number between 1 and 12, and the program uses a switch statement to display the name of the month. This program demonstrates basic input/output operations and the use of switch-case statements in C++.




ARRAY SUMMATION

This C++ program prompts the user to enter nine numbers, stores them in an array, and then calculates the sum of each number with itself. The results are stored in a separate array and displayed to the user. The program demonstrates basic array manipulation, input/output operations, and simple arithmetic in C++.
